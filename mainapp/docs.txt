===== Создание работы
POST
Только авторизованные пользователи
/api/v1/artwork/create/

Body:
artwork: 	[файл]
category:	<id категории>
name:		Наименование работы
description:	Описание работы

===== Получение работы по ID
GET
Только авторизованные пользователи
/api/v1/artwork/read/<id работы>/


===== Удаление работы
DEL
Только авторизованный автор работы и суперпользователь
/api/v1/artwork/read/<id работы>/
* работа не удаляется, ставится status = Del

===== Обновление работы
PATCH
Только авторизованный автор работы и суперпользователь
/api/v1/artwork/read/<id работы>/

* я не ограничивал редактируемые поля. Очевидно,можно обновлять наименоваени, описание


===== Список всех работ
GET
Все пользователи
/api/v1/artwork/list/all/
